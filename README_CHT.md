# WebP Viewer - Powered by Google Gemini

一個輕量、流暢且功能強大的網頁版 WebP 圖片檢視器。無需上傳，直接存取本地資料夾，並提供專業級的預覽與投影片功能。

## ✨ 特色功能

### 🛠 核心操控
* **本地資料夾存取**：利用現代瀏覽器 API，直接選取並讀取本地資料夾中的所有 WebP 檔案。
* **快速載入 (IndexedDB)**：自動記住上次開啟的資料夾路徑，下次開啟時一鍵即可恢復載入。
* **預設比例鎖定**：系統預設鎖定圖片長寬比，調整寬度時高度會自動同步。
* **靈活縮圖調整**：支援即時調整縮圖寬高（預設加大至 300px），適應各種螢幕解析度。

### 📊 排序與搜尋
* **多樣化排序模式**：支援「預設、檔名、資料夾路徑、日期、檔案大小」排序。
* **智慧隨機打亂**：一鍵隨機打亂圖片順序，並自動切換至隨機模式。
* **直覺排序切換**：使用單鍵「↑ / ↓」即可切換遞增或遞減。

### 🖼 進階燈箱 (Lightbox)
* **流暢縮放**：支援滑鼠滾輪縮放與滑鼠拖拽位移，精準檢視細節。
* **自動投影片**：可自定義切換速度，並支援播放按鈕的顯示/隱藏。
* **全螢幕模式**：一鍵進入沉浸式瀏覽體驗。

## ⌨️ 鍵盤快捷鍵

為了提升操作效率，本檢視器在**燈箱開啟狀態**下支援以下快捷操作：

| 按鍵 | 功能說明 |
| :--- | :--- |
| **空格鍵 (Space)** | **開始 / 暫停** 自動播放投影片 |
| **右方向鍵 (→)** | 切換至 **下一張** 圖片 (並自動停止播放) |
| **左方向鍵 (←)** | 切換至 **上一張** 圖片 (並自動停止播放) |
| **ESC** | **關閉燈箱** 並返回藝廊畫面 |

## 🚀 快速上手

1.  **下載檔案**：下載 `webp_viewer2.html` 到你的電腦。
2.  **開啟程式**：使用 Chrome、Edge 或 Brave 瀏覽器直接開啟此 HTML 檔案。
3.  **選擇資料夾**：點擊左上角的 **📂 選擇**，授權瀏覽器存取資料夾。

## ⚠️ 瀏覽器支援

由於本專案使用了 **File System Access API**，請使用以下瀏覽器以獲得最佳體驗：
* **Google Chrome** (建議)
* **Microsoft Edge**
* **Brave** (如失效請關閉 Shields 或開啟相關 Flags 設定)

## 🛠 技術細節

* **HTML5 / CSS3**：響應式 Grid 佈局與現代化 UI 開關。
* **Vanilla JavaScript**：無外部框架依賴，確保輕量與高效執行速度。
* **IndexedDB**：用於持久化儲存資料夾控制代碼 (Folder Handles)。
