# Personal WebP Viewer 🚀

這是一個專為高效瀏覽 WebP 圖片設計的個人化輕量網頁工具。無需上傳圖片到伺服器，直接透過瀏覽器安全地存取本地資料夾，並具備智慧目錄管理與專業級燈箱功能。

## ✨ 特色功能

### 📁 智慧目錄管理
* **目錄樹選擇 (Tree View)**：選取資料夾後即可預覽完整目錄結構，精確勾選想要掃描的子目錄。
* **層級聯動勾選**：智慧核取邏輯，勾選父資料夾時自動全選/取消所有子目錄。
* **變更範圍 (Reselect Scope)**：無需重新開啟系統檔案選擇器，即可從目前已開啟的資料夾中重新定義掃描範圍。
* **快速載入 (Quick Load)**：利用 IndexedDB 記憶權限，一鍵對上次使用的資料夾執行深度遞迴掃描。
* **智慧檔案過濾**：自動排除 macOS 產生的 `._` 系統檔與所有隱藏檔案 (`.`)。

### 🛠 核心操控
* **預設比例鎖定**：系統預設鎖定長寬比，調整寬度時高度自動同步，維持檢視美觀。
* **動態縮圖尺寸**：支援即時調整縮圖寬高（預設加大至 300px），適應各種螢幕解析度。
* **專業排序邏輯**：
    * 支援「預設、隨機、檔名、資料夾、日期、檔案大小」排序。
    * 一鍵切換「↑ / ↓」遞增或遞減。
    * **智慧打亂**：隨機排列圖片並自動鎖定隨機模式，防止排序衝突。

### 🖼 進階燈箱 (Lightbox)
* **流暢縮放與拖曳**：支援滑鼠滾輪縮放與左鍵拖曳位移，檢視細節不吃力。
* **自動投影片**：可自定義播放速度（0.5s - 15s），並提供現代化 Switch 開關切換播放鈕顯示。
* **全螢幕模式**：一鍵進入沉浸式瀏覽環境。

---

## ⌨️ 鍵盤快捷鍵 (燈箱開啟時)

| 按鍵 | 功能說明 |
| :--- | :--- |
| **空格鍵 (Space)** | **開始 / 暫停** 自動播放 |
| **右方向鍵 (→)** | 切換至 **下一張** 圖片 |
| **左方向鍵 (←)** | 切換至 **上一張** 圖片 |
| **ESC** | **關閉燈箱** 並返回藝廊 |

## 🚀 快速上手

1. **取得檔案**：將 `PersonalWebPViewer.html` 儲存至本地。
2. **開啟程式**：使用 Chrome 或 Edge 瀏覽器直接開啟 HTML 檔案。
3. **授權權限**：點擊 **📂 選擇資料夾** 並授權瀏覽器讀取權限。
4. **精確掃描**：在彈出的視窗中勾選欲檢視的目錄。
